# Core

El Core contiene las capacidades compartidas por todos los componentes del Tefi Design System.

Su objetivo es reutilizar al máximo las capacidades de la plataforma (React, HTML, CSS, TypeScript, JavaScript y Browser APIs) para construir componentes pequeños, consistentes y fáciles de mantener.

El Core no contiene componentes visuales.

El Core proporciona las herramientas necesarias para construirlos.

---

## Filosofía

Un elemento entra al Core únicamente cuando:

- Se utiliza en varios componentes.
- Reduce duplicación.
- Simplifica la lectura.
- Aprovecha una capacidad existente de la plataforma.
- Aporta valor real al Design System.
