@tefi/design-system

Sistema de diseño React desarrollado con TypeScript.

Instalación

npm install @tefi/design-system

Uso

import { Button } from "@tefi/design-system";
import "@tefi/design-system/styles.css";

export default function Example() {
  return (
    <Button>
      Hola mundo
    </Button>
  );
}

Requisitos

- React 18+
- React DOM 18+
